# Blog for spring boot practice

Simple Blog 個人化部落格 side project

## System

1. Back-end：Spring boot

2. Front-end：html、css、js

## 簡單部落格特色

1. 部落格 - 類型/標籤分類

2. 可搜尋文章

3. 後台管理 - 登入/登出系統

4. 可用Markdown語法

5. 可回覆文章 - 有留言區

6. 後台系統 - 可增刪改查文章/標籤

7. 預計未來找機會部署於雲端 vps or docker
